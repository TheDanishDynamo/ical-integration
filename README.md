# ical-integration
Utility or module to parse ical file and produce json and other output

# Objective

## Parser - Read Only
Develop a ical parser that will produce a developer friendly output. 

## Wrapper - Read Write
Develop a developer wrapper that can be used to navigate an ical file and make updates to the file
